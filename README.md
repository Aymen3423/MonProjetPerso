Projet — Vérification Nom et Âge en C#
Description
Ce programme demande à l’utilisateur :

Son prénom — s’il est vide ou null, le programme affiche Invalid et s’arrête.

Son âge — s’il est mineur (moins de 18 ans), il propose de quitter le programme.

Technologies utilisées

Langage : C#

IDE recommandé : Visual Studio


Instalation 

git clone https://github.com/Aymen3423/MonProjetPerso/
et lance le programme sur Visual Studio
